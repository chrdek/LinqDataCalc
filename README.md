This nuget package implements basic statistics and data analytics functions. 

__Some of them are displayed in the table below:__
<br/>

<br/>

|  Function  |  Explanation  |
|:------------:|:------------:|
|  STDEVP()    | Standard Deviation Calc.             |
|  MODE()      | Mode Value             |
|  MEDIAN()    | Median Calc.              |
